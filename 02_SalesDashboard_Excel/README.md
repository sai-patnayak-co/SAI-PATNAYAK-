# 02_SalesDashboard_Excel

Short description
- Build an Excel-based sales dashboard showing KPIs, trends, and top products/customers.

Objective
- Create an interactive, refreshable dashboard to monitor sales performance and surface trends.

Data
- Example file: data/sales_data.xlsx
- Typical columns: date, product_id, product_name, region, sales_amount, quantity, customer_id

Steps
1. Clean and validate sales data.
2. Create structured tables and PivotTables for KPI calculations.
3. Build charts: time series, top products, regional breakdown, cohort/retention where applicable.
4. Add slicers / dropdowns for interactivity.
5. Document dashboard usage and data refresh instructions.

Deliverables
- Dashboard Excel file (with PivotTables/Charts).
- KPI definitions and calculation sheet.
- README describing interpretation and refresh steps.
