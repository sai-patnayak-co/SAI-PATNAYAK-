# 11_BA_Communication

Short description
- Templates and best practices for business analyst communication with stakeholders and engineering teams.

Objective
- Improve clarity and consistency in requirement handoffs, status updates, and executive communications.

Contents
- Email templates, meeting agenda templates, requirement handoff checklist, acceptance criteria template.

Steps
1. Assemble templates and examples for common scenarios.
2. Create a requirement readiness and handoff checklist.
3. Provide examples of concise executive summaries and status updates.

Deliverables
- Folder of templates (DOCX/MD) and a README with guidance on usage.
