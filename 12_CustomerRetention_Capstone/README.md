# 12_CustomerRetention_Capstone

Short description
- Capstone project focused on customer retention strategies and lifecycle analysis.

Objective
- Build a complete analysis identifying drivers of churn and actionable retention strategies.

Data
- Customer transactions, subscription status, engagement logs, support tickets (sample paths in /data).

Steps
1. Define retention metrics (churn rate, cohort retention curves).
2. Engineer features and analyze drivers of churn; build predictive models where applicable.
3. Propose interventions and estimate impact (A/B test designs if possible).
4. Document implementation plan and measurement framework.

Deliverables
- Analysis notebook or SQL + BI deliverables with visuals.
- Retention playbook with prioritized initiatives.
- README with reproduction steps and required data schema.
