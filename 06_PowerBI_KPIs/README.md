# 06_PowerBI_KPIs

Short description
- Power BI report containing KPIs, drill-throughs, and interactive visuals for stakeholders.

Objective
- Build a maintainable Power BI report with clear measures and navigation for decision-making.

Data
- Data sources: CSV / SQL / Excel connections (document connections in a DataSources.md)
- Maintain a DAX measures reference file.

Steps
1. Define KPIs and data model.
2. Create DAX measures and implement time intelligence.
3. Design report pages focused on stakeholder needs.
4. Add bookmarks, tooltips, and drill-throughs for deeper exploration.
5. Document refresh configuration and dataset credentials (do not store secrets here).

Deliverables
- .pbix file (Power BI report) and DAX measures list.
- README with usage and refresh instructions.
