# 04_SQL_RetailAnalysis

Short description
- Retail sales analysis using SQL to derive insights on revenue, SKU performance, and store trends.

Objective
- Provide reusable SQL queries and a short insights report to inform merchandising and operations.

Data
- Example tables: sales, products, stores, customers, returns

Steps
1. Explore schema and assess data quality.
2. Create baseline queries for revenue by date, product, and store.
3. Compute metrics: avg. basket size, repeat purchase rate, returns rate.
4. Build segmented analyses (by channel, region, product category).
5. Document queries and include sample visualizations.

Deliverables
- Commented SQL scripts.
- Insights report (Markdown/PDF) with charts and findings.
