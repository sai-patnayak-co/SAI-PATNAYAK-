# 10_SQL+BI_InsightsReport

Short description
- Combine SQL queries and BI visualizations to produce a concise, reproducible insights report.

Objective
- Deliver a reproducible pipeline from raw data to visual insights and recommendations.

Data
- Source tables and key query excerpts documented in /sql and /data folders.

Steps
1. Extract and transform data using SQL.
2. Load results into a BI tool (Power BI / Excel).
3. Build visuals and write a narrative insights report.
4. Document reproducible steps and query parameters.

Deliverables
- SQL scripts, BI files, and an insights report with key findings and recommended actions.
- README with reproduction steps and required credentials (do not include secrets).
