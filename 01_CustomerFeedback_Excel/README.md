# 01_CustomerFeedback_Excel

Short description
- Analyze customer feedback collected in Excel to extract themes, sentiment, and actionable recommendations.

Objective
- Clean, structure, and analyze feedback to surface top themes and sentiment trends.

Data
- Example file: data/customer_feedback.xlsx
- Typical columns: customer_id, date, channel, rating, comments

Steps
1. Inspect and clean data (handle missing values, duplicates).
2. Standardize rating scales and categorize channels.
3. Perform text cleaning and basic NLP (tokenization, stopword removal).
4. Extract themes via keyword grouping or simple topic modeling.
5. Compute sentiment scores and visualize distributions.
6. Summarize recommendations and next steps.

Deliverables
- Cleaned Excel or CSV with processed columns.
- Charts / pivot tables or a dashboard (Excel or Power BI).
- This README with summary insights and recommended actions.

Notes
- Keep raw and processed files in separate folders.
- Add example pivot table screenshots for non-technical stakeholders.
