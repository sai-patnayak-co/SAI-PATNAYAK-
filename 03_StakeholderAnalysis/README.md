# 03_StakeholderAnalysis

Short description
- Map stakeholders, their interest and influence, and recommend engagement strategies.

Objective
- Produce a stakeholder map and concise engagement plan for a project or product.

Data
- Stakeholder list: name, role, department, interest_level, influence_level, communication_preference

Steps
1. Collect stakeholder information via interviews or internal sources.
2. Rate interest and influence and place stakeholders on a 2x2 matrix.
3. Define key messages and engagement cadence per stakeholder group.
4. Produce visual stakeholder map and a RACI if needed.

Deliverables
- Stakeholder map (PNG/PDF).
- Engagement plan and RACI table.
- README summarizing findings and next steps.
