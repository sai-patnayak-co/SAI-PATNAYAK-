# 05_CustomerSegmentation

Short description
- Segment customers using behavioral and demographic data to guide targeting and personalization.

Objective
- Produce multiple segmentation approaches and recommend a preferred, business-actionable segmentation.

Data
- Customer-level data: customer_id, recency, frequency, monetary, demographics, product_categories_purchased

Steps
1. Prepare RFM variables and additional features.
2. Run clustering (K-Means, hierarchical) and/or rule-based segmentation.
3. Evaluate segments for size, stability, and interpretability.
4. Profile segments and recommend targeted actions.

Deliverables
- Code or SQL used for segmentation (notebooks or scripts).
- Segment profiles, size, and recommended campaigns.
- README documenting methodology and reproduction steps.
